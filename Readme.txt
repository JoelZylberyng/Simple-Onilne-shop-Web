To install the app:
1. Run command-line as administrator
2. cd to "...\web app" and run the command "npm install"
3. Run command "nodemon start" or "npm start"
4. In a web broswer, go to address "localhost:3000"

To install the database:
1. Install MongoDB locally
2. copy "db\cellShop" to the MongoDB folder ".\bin\dump"
3. At ".\bin" run command-line as administrator and run the command "mongorestore"
